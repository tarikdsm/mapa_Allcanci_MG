# Documentação do Projeto

Este diretório concentra a documentação operacional e de continuidade do `Mapa Allcanci MG`.

## Ordem de leitura recomendada

1. [`HANDOFF_Codex.md`](./HANDOFF_Codex.md)
2. [`ARQUITETURA.md`](./ARQUITETURA.md)
3. [`FONTES_E_DADOS.md`](./FONTES_E_DADOS.md)
4. [`OPERACAO_E_DEPLOY.md`](./OPERACAO_E_DEPLOY.md)
5. [`HISTORICO_DO_PROJETO.md`](./HISTORICO_DO_PROJETO.md)

## Objetivo da documentação

- permitir que uma pessoa nova entenda o projeto rapidamente
- permitir que outro Codex continue o trabalho sem depender do histórico do chat
- registrar decisões técnicas e operacionais
- preservar o histórico do que já foi construído

## O que cada arquivo cobre

- `HANDOFF_Codex.md`: estado atual, decisões, pendências e próximos passos
- `ARQUITETURA.md`: visão técnica da aplicação e fluxo de dados
- `FONTES_E_DADOS.md`: origem dos dados, formatos e regras de atualização
- `OPERACAO_E_DEPLOY.md`: setup local, execução, rebuild e publicação
- `HISTORICO_DO_PROJETO.md`: linha do tempo consolidada do projeto
